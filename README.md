# Nuestro Hogar 🏠

PWA de gestión de tareas del hogar para pareja, con gamificación (XP, niveles, avatares evolutivos, wallet canjeable, catálogo de premios mutuos).

## Stack
- HTML/CSS/JS vanilla (un único archivo `index.html`, sin build tools)
- Firebase (Firestore + Auth) para sincronización en tiempo real entre dispositivos
- Desplegado en Netlify, conectado a este repositorio para despliegue automático

## Cómo se actualiza
Cualquier cambio subido a la rama `main` de este repositorio se despliega automáticamente en Netlify en segundos. No es necesario arrastrar archivos manualmente — Claude puede subir los cambios directamente a este repositorio.

## Estructura
- `index.html` — toda la aplicación (HTML + CSS + JS en un solo archivo)
